Código Dealer 'XXXXXX' não consta no arquivo

Motivo: O código dealer (adabás) descrito na coluna A do arquivo possui uma numeração diferente do cadastrado no VIVO GO.

Solução: Abra seu arquivo (extrato fcdi), verifique na coluna A a numeração do seu adabás, ela deve estar de acordo com a numeração cadastrada no VIVO GO. Para checar no VIVO GO, acesse o menu Configurações > Rede > Pontos de Venda ( Ver todos )
